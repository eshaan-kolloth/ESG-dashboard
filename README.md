# 📊 Project Overview

The **AI-Powered ESG Dashboard** is an interactive data application designed to analyze and visualize **Environmental, Social, and Governance (ESG)** insights using modern data science and AI tools.

This platform allows users to:

* Explore ESG metrics through visual dashboards
* Analyze sustainability performance indicators
* Interact with an AI chatbot for ESG-related insights
* Understand sustainability data through intuitive visualizations

The application demonstrates how **data science, AI, and interactive dashboards** can be combined to support **sustainable decision-making and ESG analysis**.

---

# 🎯 Key Features

### 📈 ESG Data Dashboard

Interactive visualizations that help users explore ESG indicators such as sustainability metrics, performance trends, and comparative insights.

### 🤖 AI ESG Chatbot

An intelligent chatbot that answers ESG-related questions and provides explanations about sustainability metrics and concepts.

### 📊 Data Visualization

Dynamic charts and graphs help users quickly interpret ESG performance and sustainability indicators.

### ⚡ Interactive Web Interface

A user-friendly interface that enables users to explore ESG data without needing technical knowledge.

# ⚙️ Installation & Setup

To run the project locally:

### 1️⃣ Clone the repository

```bash
git clone https://github.com/your-username/esg-dashboard.git
```

### 2️⃣ Navigate to the project folder

```bash
cd esg-dashboard
```

### 3️⃣ Install dependencies

```bash
pip install -r requirements.txt
```

### 4️⃣ Run the Streamlit application

```bash
streamlit run streamlit_app.py
```
